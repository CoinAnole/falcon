# Flux 2

> Text-to-image generation with FLUX.2 [dev] from Black Forest Labs. Enhanced realism, crisper text generation, and native editing capabilities.

## Variants

Flux 2 comes in three variants optimized for different use cases:

| Variant | Endpoint | Model ID | Description |
|---------|----------|----------|-------------|
| **Flux 2** (Standard) | `https://fal.run/fal-ai/flux-2` | `fal-ai/flux-2` | Full quality generation with maximum control |
| **Flux 2 Flash** | `https://fal.run/fal-ai/flux-2/flash` | `fal-ai/flux-2/flash` | Fastest generation at lowest cost — in a flash |
| **Flux 2 Turbo** | `https://fal.run/fal-ai/flux-2/turbo` | `fal-ai/flux-2/turbo` | Balanced speed and quality — at turbo speed |

All variants share the same core capabilities but differ in generation speed, cost, and available parameters.

## Overview

- **Category**: text-to-image
- **Kind**: inference


## API Information

This model can be used via our HTTP API or more conveniently via our client libraries.
See the input and output schema below, as well as the usage examples.


### Input Schema

The API accepts the following input parameters:


- **`prompt`** (`string`, _required_):
  The prompt to generate an image from.
  - Examples: "A cinematic portrait of a weathered sailor with white beard, golden hour lighting, dramatic shadows on face, shallow depth of field, photorealistic"

- **`guidance_scale`** (`float`, _optional_):
  Guidance Scale is a measure of how close you want the model to stick to your prompt when looking for a related image to show you.
  - Default: `2.5`
  - Range: `0` to `20`

- **`seed`** (`integer`, _optional_):
  The seed to use for the generation. If not provided, a random seed will be used.

- **`num_inference_steps`** (`integer`, _optional_, **Flux 2 Standard only**):
  The number of inference steps to perform. Not available on Flash/Turbo variants.
  - Default: `28`
  - Range: `4` to `50`

- **`image_size`** (`ImageSize | Enum`, _optional_):
  The size of the image to generate. The width and height must be between 512 and 2048 pixels.
  - Default: `"landscape_4_3"`
  - One of: ImageSize | Enum

- **`num_images`** (`integer`, _optional_):
  The number of images to generate.
  - Default: `1`
  - Range: `1` to `4`

- **`acceleration`** (`AccelerationEnum`, _optional_, **Flux 2 Standard only**):
  The acceleration level to use for the image generation. Not available on Flash/Turbo variants.
  - Default: `"regular"`
  - Options: `"none"`, `"regular"`, `"high"`

- **`enable_prompt_expansion`** (`boolean`, _optional_):
  If set to true, the prompt will be expanded for better results.
  - Default: `false`

- **`sync_mode`** (`boolean`, _optional_):
  If `True`, the media will be returned as a data URI and the output data won't be available in the request history.
  - Default: `false`

- **`enable_safety_checker`** (`boolean`, _optional_):
  If set to true, the safety checker will be enabled.
  - Default: `true`

- **`output_format`** (`OutputFormatEnum`, _optional_):
  The format of the generated image.
  - Default: `"png"`
  - Options: `"jpeg"`, `"png"`, `"webp"`



**Required Parameters Example**:

```json
{
  "prompt": "A cinematic portrait of a weathered sailor with white beard, golden hour lighting, dramatic shadows on face, shallow depth of field, photorealistic"
}
```

**Full Example (Flux 2 Standard)**:

```json
{
  "prompt": "A cinematic portrait of a weathered sailor with white beard, golden hour lighting, dramatic shadows on face, shallow depth of field, photorealistic",
  "guidance_scale": 2.5,
  "num_inference_steps": 28,
  "image_size": "landscape_4_3",
  "num_images": 1,
  "acceleration": "regular",
  "enable_safety_checker": true,
  "output_format": "png"
}
```

**Full Example (Flux 2 Flash/Turbo)**:

```json
{
  "prompt": "A cinematic portrait of a weathered sailor with white beard, golden hour lighting, dramatic shadows on face, shallow depth of field, photorealistic",
  "guidance_scale": 2.5,
  "image_size": "landscape_4_3",
  "num_images": 1,
  "enable_safety_checker": true,
  "output_format": "png"
}
```


### Output Schema

The API returns the following output format for all variants:

- **`images`** (`list<ImageFile>`, _required_):
  The generated images
  - Array of ImageFile
  - Examples: [{"url":"https://storage.googleapis.com/falserverless/example_outputs/flux2_dev_t2i_output.png"}]

- **`timings`** (`Timings`, _required_)

- **`seed`** (`integer`, _required_):
  Seed of the generated Image. It will be the same value of the one passed in the
  input or the randomly generated that was used in case none was passed.

- **`has_nsfw_concepts`** (`list<boolean>`, _required_):
  Whether the generated images contain NSFW concepts.
  - Array of boolean

- **`prompt`** (`string`, _required_):
  The prompt used for generating the image.



**Example Response**:

```json
{
  "images": [
    {
      "url": "https://storage.googleapis.com/falserverless/example_outputs/flux2_dev_t2i_output.png"
    }
  ],
  "prompt": ""
}
```


## Usage Examples

### cURL

**Flux 2 (Standard)**

```bash
curl --request POST \
  --url https://fal.run/fal-ai/flux-2 \
  --header "Authorization: Key $FAL_KEY" \
  --header "Content-Type: application/json" \
  --data '{
     "prompt": "A cinematic portrait of a weathered sailor with white beard, golden hour lighting, dramatic shadows on face, shallow depth of field, photorealistic"
   }'
```

**Flux 2 Flash**

```bash
curl --request POST \
  --url https://fal.run/fal-ai/flux-2/flash \
  --header "Authorization: Key $FAL_KEY" \
  --header "Content-Type: application/json" \
  --data '{
     "prompt": "An army of ants battles for a sugar cube in a giant sandbox, with tiny plastic soldier toys caught in the melee. Ant bodies shine against gritty beige sand and the glossy, crystalline cube."
   }'
```

**Flux 2 Turbo**

```bash
curl --request POST \
  --url https://fal.run/fal-ai/flux-2/turbo \
  --header "Authorization: Key $FAL_KEY" \
  --header "Content-Type: application/json" \
  --data '{
     "prompt": "A realistic photograph of a vintage typewriter with a sheet of paper inserted that says 'Chapter One: The Journey Begins,' sunlight falling across the desk."
   }'
```

### Python

Ensure you have the Python client installed:

```bash
pip install fal-client
```

Then use the API client to make requests:

**Flux 2 (Standard)**

```python
import fal_client

def on_queue_update(update):
    if isinstance(update, fal_client.InProgress):
        for log in update.logs:
           print(log["message"])

result = fal_client.subscribe(
    "fal-ai/flux-2",
    arguments={
        "prompt": "A cinematic portrait of a weathered sailor with white beard, golden hour lighting, dramatic shadows on face, shallow depth of field, photorealistic"
    },
    with_logs=True,
    on_queue_update=on_queue_update,
)
print(result)
```

**Flux 2 Flash**

### JavaScript

Ensure you have the JavaScript client installed:

```bash
npm install --save @fal-ai/client
```

Then use the API client to make requests:

**Flux 2 (Standard)**

```javascript
import { fal } from "@fal-ai/client";

const result = await fal.subscribe("fal-ai/flux-2", {
  input: {
    prompt: "A cinematic portrait of a weathered sailor with white beard, golden hour lighting"
  },
  logs: true,
  onQueueUpdate: (update) => {
    if (update.status === "IN_PROGRESS") {
      update.logs.map((log) => log.message).forEach(console.log);
    }
  },
});
console.log(result.data);
console.log(result.requestId);
```

**Flux 2 Flash**

